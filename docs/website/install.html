Installation

One command. No compilation. No Fortran. No MPI configuration.

From PyPI (recommended)
pip install neurocgmd
Installs the neurocgmd CLI and all packages. 
pip install neurocgmd
From source (development)
git clone + pip install -e
Editable mode for development. 
git clone https://github.com/sciencemaths-collab/neurocgmd.git
cd neurocgmd
pip install -e ".[dev]"
verify your installation
▶ VERIFY
neurocgmd info       # color-coded capabilities display\nneurocgmd --version  # neurocgmd 1.0.0
