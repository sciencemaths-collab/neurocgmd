Running Simulations

The NeuroCGMD command-line interface. Click any command to expand.

▶ COMMANDS
neurocgmd run config.toml
Full pipeline
Prepare + simulate + back-map + analyze + plot. One command does everything.
neurocgmd prepare config.toml
Preparation only
Import PDB, build CG topology, generate prepared_bundle.json.
neurocgmd analyze config.toml
Analyze existing data
Run analysis pipeline on existing trajectory. Can run on partial data while simulation continues.
neurocgmd info
Platform capabilities
Color-coded display of all layers, integrators, analysis tools, supported systems.
the run pipeline
▶ PIPELINE STAGES — neurocgmd run executes all 7 stages in order
1. Import
Parse PDB, identify chains, map to CG beads
2. Topology
Build bonds, angles, nonbonded lists
3. NVT
Thermostat coupling at constant volume
4. NPT
Pressure coupling at constant pressure
5. Production
Hybrid dynamics: CG + QCloud + ML
6. Back-map
CG trajectory to AA coordinates
7. Analyze
Compute all observables, generate 20+ plots
output directory
▶ OUTPUT STRUCTURE
outputs/my_system/\n  prepared_bundle.json          # Topology & parameters\n  energies.csv                  # Energy time series (live)\n  traj.jsonl                    # CG trajectory (JSONL)\n  run_summary.json              # Run metadata\n  cg_trajectory.pdb             # CG multi-model PDB\n  aa_backmapped_trajectory.pdb  # AA multi-model PDB\n  aa_reference.pdb              # Crystal structure\n  plots/                        # 20+ analysis PNGs
