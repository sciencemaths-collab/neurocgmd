four layers fuse into one cooperative simulation
▶ THE FOUR LAYERS — click any card to expand
CG Dynamics
Speed: ~1000 steps/min
Langevin integration with classical forcefields. BAOAB splitting, velocity-Verlet, O(N) neighbor lists.
QCloud Quantum
Accuracy: AA-level physics
Force corrections on priority regions with adaptive focusing. Event detection for bond forming/breaking.
ML Residual
Efficiency: fewer QCloud calls
Neural network learns correction patterns on-the-fly from QCloud. Reduces quantum compute over time.
Back-mapping
Detail: real residues
Reconstructs full all-atom coordinates from CG trajectory. Carries CG + QCloud + ML information through.
the result
Ftotal = FCG + ΔFQCloud + α ΔFML  →  integrator  →  CG positions  →  back-map  →  AA
