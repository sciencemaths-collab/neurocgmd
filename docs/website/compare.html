How NeuroCGMD Compares

Feature comparison with established molecular dynamics engines.

NeuroCGMD
500 KB, pip install
Pure Python. CG+QM+ML cooperative. Auto analysis. Integrated back-mapping.
GROMACS
~50 MB, compile
C/C++/CUDA. Industry standard. Excellent performance. Requires separate tools for analysis.
OpenMM
~200 MB, conda
C++/Python hybrid. GPU-accelerated. Plugin architecture for ML.
NAMD
~100 MB, compile
C++/Charm++. Scalable parallel. MPI required.
